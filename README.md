# Retail Sales Data Analysis using SQL

## Project Description
This project analyzes retail sales data using SQL to identify sales trends, top-selling products, and customer purchasing behavior.

## Tools Used
SQL (MySQL)

## Dataset Columns
- order_id
- order_date
- customer_name
- city
- product_name
- category
- quantity
- sales

## Key Analysis
- Top selling products
- Category wise sales
- City wise sales performance

## SQL Queries Used
- GROUP BY
- ORDER BY
- SUM
- COUNT
